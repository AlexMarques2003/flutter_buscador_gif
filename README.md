# flutter_buscador_gif

Aplicativo de busca de gifs.

## Como testar

Baixe o visual studio code
Instale as extensões dart e flutter
Execute o emulador
Abra o arquivo main.dart
Clique em Debug / Iniciar Depuração / Aponte para o emulador e pronto
